# 06. UI仕様

## デザイン方針

- **業務ツール**: 派手な装飾は不要、情報密度を優先
- **shadcn/ui ベース**: コンポーネントは原則shadcn/uiを使う
- **モバイル対応**: タブレット程度までは考慮、スマホは将来対応
- **配色**: `2E5984`（ブルーグレー、業務ツールらしい落ち着いた色）をプライマリに
- **フォント**: 数値は `tabular-nums` で桁揃え必須

**フロー**:

2. アップロード進捗表示（プログレスバー）

**バリデーション**: PDF以外、10MB超、画像PDFは弾く。

## レスポンシブ対応

- デスクトップ（1280px+）: 上記レイアウトをそのまま
- タブレット（768-1279px）: サイドバーを折りたたみ可能に
- モバイル（〜767px）: ハンバーガーメニュー、テーブルは横スクロール

## アクセシビリティ

- キーボード操作（Tab, Enter, Escape）対応
- スクリーンリーダー用のaria属性
- カラーコントラストはWCAG AA準拠

## ローディング・エラー状態

- データ取得中: shadcn/uiの `Skeleton`
- エラー: `Alert` コンポーネントで表示
- 空状態: アイコン＋メッセージ＋次のアクションボタン

## トースト通知

- アップロード成功・失敗
- 担当者割り当て完了
- メモ保存
- 各種エラー

`sonner` ライブラリを推奨。
